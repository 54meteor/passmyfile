# 隔空投送 - MVP 开发计划

## 1. 项目概述

**项目名称**: 隔空投送 (隔空投送精简版)
**项目定位**: 跨平台局域网文件传输工具
**技术栈**: Tauri 2.x + React + TypeScript + Rust

## 2. 功能范围 (MVP)

### 2.1 核心功能

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 设备发现 | P0 | 局域网内自动发现在线设备 |
| 设备列表 | P0 | 显示设备名称、IP、在线状态 |
| 文件传输 | P0 | 点对点直连传输文件 |
| 传输进度 | P1 | 显示传输速度、进度百分比 |
| 接收确认 | P1 | 接收方确认后才保存文件 |

### 2.2 非功能需求

- 支持 Windows/macOS/Linux
- 单文件大小支持到 1GB
- 传输过程可中断、可续传

### 3.2 网络协议

```
发现协议 (UDP):
  - 广播地址: 255.255.255.255
  - 端口: 18765
  - 方式: 每5秒广播一次自身信息
  - 监听: 收到广播后记录设备

传输协议 (TCP):
  - 端口: 18766
  - 连接: 发起方主动连接接收方
  - 流程:
    1. 发送方发起连接
    2. 发送文件元信息 (文件名、大小)
    3. 接收方确认接收
    4. 开始传输文件数据
```

### 3.3 消息格式

```rust
// 设备发现消息 (UDP)
struct DiscoveryMsg {
    device_id: String,      // 设备唯一ID (UUID)
    device_name: String,   // 设备名称 (可自定义)
    ip: String,           // IP地址
    port: u16,            // TCP传输端口
    version: String,      // 协议版本
}

// 文件传输请求 (TCP)
struct FileTransferRequest {
    file_id: String,      // 传输ID
    file_name: String,    // 文件名
    file_size: u64,       // 文件大小(字节)
    sender_id: String,    // 发送方ID
}

// 文件数据块
struct FileChunk {
    chunk_id: u32,        // 块序号
    data: Vec<u8>,       // 数据 (最大 64KB)
}
```
